# MR 資材部門管理系統 - 作業總盤點報告

## 1. MR 模組總覽

- **MR 模組定位**：負責企業資材 (Material Requirements & Inventory) 管理。涵蓋材料基本資料建立、材料採購與調度、入庫與發料、庫存查詢及進銷存報表等。
- **資材部門主要負責範圍**：
  1. 維護材料編碼與基本分類 (顏色、厚度、幅度、紋路等)。
  2. 建立與維護料號主檔 (MR035)。
  3. 執行樣品或量產所需之材料採購作業 (MR045)。
  4. 管控倉庫的材料入出庫與異動 (MR105, MR110, MR115)。
  5. 執行庫存盤點與報表統計。
- **與其他模組的關聯概述**：
  - **BA (共用基本資料)**：資材供應商需參照 BA 模組的廠商主檔；幣別、單位等依賴 BA 共用代碼。
  - **DP (開發部門)**：樣品開發的 BOM 依賴 MR 建立的料號主檔；開發階段可能需要發料/調料。
  - **SM (庫存管理)**：MR 負責的材料庫存為整體系統庫存的一環，出入庫異動資料需與 SM 同步。
  - **FA (財務應付帳款)**：材料採購與驗收將觸發後續的應付帳款與成本計算，與財務 FA 模組有關。

## 3. Pattern 分類統計

- **Pattern A (單表 CRUD / 基本資料)**：7 支 (MR001, MR002, MR010, MR020, MR025, MR030, MR031)
- **Pattern A2 (上列表下編輯)**：1 支 (MR035 - 初判可能為 A2 或 B)

- **Pattern R (報表/查詢)**：4 支 (MR040, MR053, MR130, MR135)
- **Pattern Special (特殊流程/待確認)**：5 支 (MR111, MR116, MR125, MR145, MR336)
- **總計**：23 支作業

## 7. 依賴與關聯

MR 模組與其他模組之間有緊密耦合：
- **BA**：MR045 (採購單) 需 Lookup 供應商、付款條件、幣別；所有基礎材料設定均可能依賴 BA 共用代碼。
- **DP**：DP 樣品 BOM 需 Lookup MR035 料號主檔。MR110 發料可能直接發給 DP 單位。
- **SM (庫存)**：入出庫單據 (MR105, MR110, MR115) 需同步更新庫存餘額。
- **FA (財務)**：MR105 驗收後將產生應付帳款；MR115 退貨將沖銷應付帳款。
